# PostNia
یک ابزار ساده و سبک برای تست و مدیریت درخواست‌های HTTP است که امکان ذخیره، ویرایش، گروه‌بندی و تبدیل درخواست‌ها به Postman Collection را فراهم می‌کند.

## ویژگی‌ها

- ارسال درخواست‌های HTTP
- ذخیره و مدیریت درخواست‌ها
- دسته‌بندی درخواست‌ها در Collection
- Export به Postman Collection
- Import از Postman Collection
- اجرای بسیار سبک بدون دیتابیس
- ذخیره داده‌ها در یک فایل JSON
- قابلیت build به باینری مستقل برای Linux / Windows / Mac
